## 💻 How to Run the Code

1. Install **Anaconda** (recommended for beginners): [https://www.anaconda.com/](https://www.anaconda.com/)  
2. Open **Jupyter Notebook**  
3. Clone or download this repository  
4. Open the desired notebook  
5. Run each cell (`Shift + Enter`) to see the output
